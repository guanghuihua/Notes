# Algebraic Topology Notes

Notes, code, and visualizations for learning algebraic topology, with a focus on simplicial homology.

## Contents

- `topology_homology.ipynb`: an interactive notebook introducing chain groups, boundary maps, cycles, boundaries, homology groups, Smith normal form, and classical examples.
- `simplicial.py`: a minimal implementation of finite simplicial complexes and integer homology computation.
- `viz.py`: plotting helpers for simplicial complexes, gluing diagrams, and 3D intuition pictures.
- `build_notebook.py`: script used to generate the notebook.
- `TopologyVault/`: an Obsidian vault for connected concept notes on algebraic topology.

## Examples Covered

- Circle `S1`
- Disk `D2`
- Sphere `S2`
- Torus `T2`
- Klein bottle
- Projective plane `RP2`

## Quick Start

Run the self-check:

```bash
python3 simplicial.py
```

Open the notebook:

```bash
jupyter notebook topology_homology.ipynb
```

Open the Obsidian notes by choosing `TopologyVault/` as a vault in Obsidian.
