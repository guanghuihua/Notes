## 原帖主旨

这篇帖子想解释的是：abstract algebra、homological algebra、commutative algebra、algebraic topology 不是四块互不相干的知识，而是围绕“对象、运算、结构、映射、等价与不变量”逐层发展出来的一套语言。

一个很粗略的关系可以写成：

```text
abstract algebra
    -> groups, rings, fields, modules
    -> commutative algebra
    -> homological algebra
    -> algebraic topology
```

其中 abstract algebra 提供基本对象和基本语言；commutative algebra 深入研究 commutative rings 与 modules；homological algebra 研究 chain complexes、exact sequences 以及由此产生的 algebraic invariants；algebraic topology 则把 topological spaces 转换成 groups、rings、modules、chain complexes 等代数对象来研究。

## 1. Abstract Algebra：从具体数到抽象结构

初等代数里，我们熟悉的是数字、变量和公式，例如：

$$
a + b,\quad ab,\quad a^2 + b^2.
$$

进入 abstract algebra 之后，我们不再执着于 $a,b$ 具体是什么，而是关心：

- 有哪些 operations？
- 这些 operations 满足哪些 laws？
- 哪些 properties 在不同对象之间保持不变？

例如：

- 一个 operation 加上 associativity、identity、inverse，得到 group 的思想。
- 两个 operations，类似 addition 和 multiplication，得到 ring 的思想。
- 如果 multiplication commutes，就进入 commutative ring 的世界。
- 如果一个 abelian group 可以被 ring 中元素“标量乘”，就得到 module。

所以 abstract algebra 的关键不是“算具体数”，而是抽象出运算规律。

## 2. Groups：从“元素”到“变换”

帖子里一个重要观点是：很多代数结构可以从 transformations 的角度理解。

给定一个集合，它上面的 invertible transformations 在 composition 下形成 group。

这说明 group 不只是“某种带乘法的集合”，也可以理解为：

```text
研究 symmetry / transformation / action 的语言
```

这对 Galois theory 和 algebraic topology 都很重要：

- Galois group 研究 polynomial roots 的 symmetry。
- fundamental group 研究 loops 的 homotopy classes。
- group action 描述一个 group 如何作用在某个 set、space 或 algebraic object 上。

## 3. Rings and Modules：从数域到更一般的代数环境

如果说 vector space 是“可以做 linear algebra 的空间”，那么 module 就是 vector space 的推广：

```text
vector space: scalars come from a field
module: scalars come from a ring
```

也就是说：

- vector space over a field 通常性质很好；
- module over a ring 可能复杂得多；
- commutative algebra 大量研究 commutative rings and modules。

在 algebraic topology 里，chain groups 常常是 abelian groups 或 modules。比如 singular homology 里的 chain group 是由 simplices 生成的 free abelian group。

## 4. Commutative Algebra：研究 commutative rings 的内部结构

commutative algebra 主要研究 commutative rings 以及它们上面的 modules。

典型对象包括：

- polynomial rings
- ideals
- quotient rings
- localization
- modules over commutative rings

它和 algebraic geometry 联系很深，因为几何对象可以通过 functions 或 coordinate rings 来研究。

一个粗略直觉是：

```text
commutative algebra = 用 ring/module 的语言研究“代数化的空间”
```

例如，一个几何对象上的 functions 可能组成一个 commutative ring；研究这个 ring 的 ideals、localizations、modules，就能反过来研究几何对象的局部和整体性质。

## 5. Homological Algebra：研究 exactness 与 chain complexes

homological algebra 关心的不是单个 group、ring 或 module，而是它们之间的 maps 组成的结构。

核心对象是 chain complex：

$$
\cdots \to C_{n+1} \xrightarrow{d_{n+1}} C_n \xrightarrow{d_n} C_{n-1} \to \cdots
$$

并且满足：

$$
d_n d_{n+1} = 0.
$$

由此可以定义 homology：

$$
H_n = \ker d_n / \operatorname{im} d_{n+1}.
$$

这个形式和你之前学的 simplicial homology 完全一致：

```text
cycles = kernel
boundaries = image
homology = cycles / boundaries
```

homological algebra 的一个核心精神是：

```text
用 kernels, images, quotients, exact sequences 来测量一个 algebraic construction 哪里“没有完全 exact”
```

典型工具包括：

- chain complex
- exact sequence
- short exact sequence
- long exact sequence
- derived functor
- Ext
- Tor

## 6. Algebraic Topology：把空间变成代数对象

algebraic topology 的基本策略是：

```text
topological space
    -> algebraic invariant
```

例如：

- path-connected components 给出 $H_0$。
- loops 给出 fundamental group。
- cycles and boundaries 给出 homology groups。
- cochains 给出 cohomology rings。

所以 algebraic topology 不是单纯“画图形”，而是把 topological questions 翻译成 algebraic questions。

比如 simplicial homology 中：

$$
C_n \xrightarrow{\partial_n} C_{n-1}
$$

这里的 $C_n$ 是 free abelian group，$\partial_n$ 是 group homomorphism，而

$$
H_n = \ker \partial_n / \operatorname{im} \partial_{n+1}
$$

就是一个 homological algebra construction。

## 四者关系图

```text
abstract algebra
  ├─ group theory
  │   ├─ symmetry
  │   ├─ group actions
  │   ├─ Galois theory
  │   └─ fundamental group
  │
  ├─ ring theory
  │   └─ commutative algebra
  │       ├─ ideals
  │       ├─ quotient rings
  │       ├─ localization
  │       └─ modules
  │
  └─ modules and homomorphisms
      └─ homological algebra
          ├─ chain complexes
          ├─ exact sequences
          ├─ Ext / Tor
          └─ homology / cohomology
              └─ algebraic topology
```

## 对我学习 algebraic topology 的意义

要理解 algebraic topology，最需要补的 abstract algebra 不是所有东西，而是以下几条主线：

1. groups and homomorphisms
2. abelian groups
3. rings and modules
4. quotient structures
5. kernels and images
6. exact sequences
7. chain complexes
8. homology and cohomology

特别是：

$$
H_n = \ker \partial_n / \operatorname{im} \partial_{n+1}
$$

这句话本身就是 abstract algebra + homological algebra 的语言。
