# 单纯复形

单纯复形可以理解成用点、线段、三角形、四面体等“积木”拼出来的拓扑空间。

## 单形

- 0-单形：点
- 1-单形：线段
- 2-单形：三角形面
- 3-单形：四面体

在当前 Python 项目里，一个单形用顶点元组表示：

```python
(0, 1)      # 一条边
(0, 1, 2)   # 一个三角形面
```

## 为什么重要

有了单纯复形，我们就可以把空间离散化，然后定义：

- [[链群]]
- [[边缘算子]]
- [[闭链]]
- [[边缘链]]
- [[同调群]]

## 例子

空心三角形可以表示圆周 [[例子-S1圆周]]：

```python
SimplicialComplex([(0,1), (1,2), (0,2)])
```

实心三角形可以表示圆盘：

```python
SimplicialComplex([(0,1,2)])
```
