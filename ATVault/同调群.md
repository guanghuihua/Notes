# 同调群

同调群定义为：

```text
H_n = Z_n / B_n = ker ∂_n / im ∂_{n+1}
```

其中：

- `Z_n` 是 [[闭链]]
- `B_n` 是 [[边缘链]]

## 直觉

同调群在数：

```text
闭合了，但填不掉的东西
```

不同维度看不同类型的洞：

- `H_0`：连通分支
- `H_1`：一维环洞
- `H_2`：二维空腔

## 例子

[[例子-S1圆周]]：

```text
H_0 = Z
H_1 = Z
```

[[例子-T2环面]]：

```text
H_0 = Z
H_1 = Z + Z
H_2 = Z
```

## 如何定位洞

同调群先告诉你有几个独立洞。要定位，需要找同调类的代表闭链，然后把这些链画出来。

例如 `H_1 = Z^2` 时，可以找两个独立的 1-闭链生成元，它们分别代表两个不同的环洞。
