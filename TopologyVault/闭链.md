# 闭链

闭链是边界为 0 的链：

```text
Z_n = ker ∂_n
```

也就是说，一个 `n` 维链 `c` 是闭链，当且仅当：

```text
∂_n c = 0
```

## 直觉

- 1-闭链：首尾相接的圈
- 2-闭链：没有边界的封闭面壳

比如三角形边界：

```text
[0,1] + [1,2] - [0,2]
```

它没有端点，所以是 1-闭链。

## 如何寻找闭链

寻找所有闭链，就是解线性方程：

```text
∂_n x = 0
```

严格做整数同调时，要找整数核；这通常用 [[Smith标准形]] 或整数线性代数。

## 和同调的关系

[[同调群]] 不只是看闭链，而是看闭链里哪些不能被填掉：

```text
H_n = Z_n / B_n
```
