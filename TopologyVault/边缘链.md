# 边缘链

边缘链是某个更高维链的边界：

```text
B_n = im ∂_{n+1}
```

也就是说，`c` 是 `n` 维边缘链，当且仅当存在一个 `(n+1)` 维链 `d`，使得：

```text
c = ∂_{n+1} d
```

## 为什么边缘链一定是闭链

因为：

```text
∂_n c = ∂_n ∂_{n+1} d = 0
```

也就是：

```text
∂ ∘ ∂ = 0
```

所以：

```text
B_n ⊆ Z_n
```

## 直觉

一个实心三角形的边界是一圈边。这个圈虽然闭合，但它可以被三角形面填掉，所以它是边缘链。

在 [[例子-S1圆周]] 里，同样的一圈边没有三角形面可以填，所以它不是边缘链，而是一个真正的 1 维洞。
