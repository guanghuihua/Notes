# 例子：S1 圆周

圆周可以用空心三角形表示：

```python
SimplicialComplex([(0,1), (1,2), (0,2)])
```

## 1-闭链

绕一圈的链：

```text
[0,1] + [1,2] - [0,2]
```

它的边界为 0，所以是 [[闭链]]。

## 为什么它不是边缘链

因为这个空间里没有三角形面 `[0,1,2]`，所以没有东西能把这圈边填掉。

因此：

```text
H_1(S^1) = Z
```

## 同调群

```text
H_0 = Z
H_1 = Z
```

这表示圆周是连通的，并且有一个独立的一维洞。
