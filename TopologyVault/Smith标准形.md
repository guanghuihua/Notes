# Smith 标准形

Smith 标准形用于从整数矩阵中读出同调群的结构，尤其是挠率。

对于整数矩阵 `A`，可以通过可逆整数行列变换化成对角形式：

```text
diag(d_1, d_2, ...)
```

并且满足：

```text
d_1 | d_2 | d_3 | ...
```

## 它在同调里做什么

[[边缘算子]] 是整数矩阵。要计算：

```text
H_n = ker ∂_n / im ∂_{n+1}
```

就需要理解整数核、整数像和商群结构。Smith 标准形正是做这件事的工具。

## 挠率

如果 Smith 标准形里出现 `d > 1`，就会产生一个：

```text
Z/d
```

例如射影平面：

```text
H_1(RP^2) = Z/2
```

这个 `Z/2` 就是一种挠率，直觉上像“绕两圈才变成边界”的扭洞。
