# 边缘算子

边缘算子 `∂` 把一个单形送到它的边界。

对于一个三角形：

```text
∂[0,1,2] = [1,2] - [0,2] + [0,1]
```

符号来自公式：

```text
∂[v_0 ... v_n] = Σ (-1)^i [v_0 ... v̂_i ... v_n]
```

## 关键恒等式

```text
∂ ∘ ∂ = 0
```

意思是：

```text
边界的边界为 0
```

这解释了为什么 [[边缘链]] 一定是 [[闭链]]。

## 矩阵视角

在代码里，边缘算子可以写成整数矩阵：

```python
d1 = K.boundary_matrix(1)
d2 = K.boundary_matrix(2)
```

找 [[闭链]] 就是求这个矩阵的核：

```text
Z_n = ker ∂_n
```
