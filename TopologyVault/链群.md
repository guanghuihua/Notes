# 链群

链群 `C_n` 是所有 `n` 维单形的整数线性组合。

比如 1-链是边的整数线性组合：

```text
c = 2[0,1] - [1,2] + 3[2,3]
```

## 直觉

可以把链想成一种“带方向和权重的几何片段集合”：

- 0-链：点的组合
- 1-链：边的组合
- 2-链：面的组合

## 和同调的关系

[[边缘算子]] 把 `C_n` 送到 `C_{n-1}`：

```text
∂_n : C_n -> C_{n-1}
```

然后：

- [[闭链]] 是边界为 0 的链
- [[边缘链]] 是更高维链的边界
- [[同调群]] 比较两者的差别
